# react-native-image-pixel

Allows to get RGBA pixel data from a raster image file

## Installation

```sh
npm install react-native-image-pixel
```

## Usage

```js
import { multiply } from 'react-native-image-pixel';

// ...

const result = await multiply(3, 7);
```
